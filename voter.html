Movie Voter Web Application

This was my final project for Harvard's CS50: Introduction to Computer Science. It's a web based application that helps groups decide what movies to watch together. I utilized a variety of different languages and libraries to create this application including Python, Flask, Jinja, HTML, CSS, and SQLite. The video above demonstrates the intended usage of the site.
